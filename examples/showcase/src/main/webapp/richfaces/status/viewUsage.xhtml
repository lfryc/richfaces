The rich:status component is an indicator of an Ajax request. It has two states: start and stop. The start state indicates that an Ajax request is in progress. When an Ajax Response is returned, the component switches to the stop state.

The rich:status component could be defined as follows:

  • As the default for a view.
  • As the default for a form.
  • Referenced explicity from the component sending the request.

This example shows a view-scoped rich:status component. It should be defined outside of any forms, and will handle all the Ajax requests within the view.
